[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's take a look at the workflow architecture. Every request starts at the Security Checkpoint, which scrubs sensitive information and intercepts unauthorized dosage changes. If safe, the query moves to the Orchestrator, which decides which specialist to call and formats the routing decision. The Router node then dynamically redirects the query to one of four specialized agents: our Medication Agent, our Meal Planner Agent, our Medical Notes Agent, or the General Agent for wellness guidance. Finally, before saving any medication schedules, the flow routes through a Human Approval Checkpoint to ensure nothing is logged without the user's explicit consent.

[3:00 — SECURITY & MCP]
(show security check and MCP logs)
Security and tool integration are foundational to CareSync. The Security Checkpoint runs PII redaction and jailbreak checks on every input, logging an audit trail to stdout. Meanwhile, our MCP server isolates external APIs, letting specialists safely search for clinical recipes and translate complex abbreviations like "bid" into "twice daily."
